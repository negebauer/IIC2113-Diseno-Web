### Servers

  - Production http://iic2113-web.herokuapp.com/
  - Staging: http://iic2113-web-staging.herokuapp.com/

### Development

Copy and run this script

```bash
git clone https://github.com/negebauer/IIC2113-Web.git web
cd web
yarn
yarn start
```
